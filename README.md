# 📅 Calendar System in C

## 📌 Overview
This is a console-based Calendar System developed using the C programming language. The application allows users to view calendars, check dates, and manage events efficiently.

---

## 🚀 Features

- 📆 Display Full Year Calendar  
- 📅 Display Specific Month  
- 📍 Find Day of a Given Date  
- 🔁 Check Leap Year  
- 📝 Add Events  
- 📋 Display All Events  

---

## 🛠️ Technologies Used

- C Programming Language
- Standard Libraries (`stdio.h`, `string.h`)

---

## 🧠 Concepts Used

- Functions & Modular Programming  
- Arrays & Structures  
- Date Calculations (Day Formula)  
- Conditional Logic  
- Looping  

---

## 📂 Project Structure

- `main()` → Menu-driven program  
- `dayNumber()` → Calculates day of a given date  
- `numberofdays()` → Returns number of days in a month  
- `displayyearcalendar()` → Prints full year calendar  
- `displayspecificmonth()` → Prints selected month  
- `checkleapyear()` → Checks leap year  
- `addevent()` → Stores event details  
- `displayevent()` → Displays all events
